An admin user can edit a server

Pre conditions:

  1. A valid admin username and password is logged

Steps:

  1. Go to the servers page: "https://effizientedemo.azurewebsites.net/Security/servers"
  2. Create server with API
  3. Go to the servers page: "https://effizientedemo.azurewebsites.net/Security/servers"
  4. Fill "key:" with the value: "553725"
  5. Fill "Name:" with the value: "Prohaska Group"
  6. Fill "Url:" with the value: "https://wide-strait.net/"
  7. Click: "Save"
  8. Success message is visible
  9. Server with the key: "553725" exists in the table
  10. The server name for the key: "553725" is: "Prohaska Group"
  11. The server url for the key: "553725" is: "https://wide-strait.net/"

Post conditions:

  1. Delete the server with API requests
